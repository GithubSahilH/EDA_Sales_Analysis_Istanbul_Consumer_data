# Retail Analytics: Customer Segmentation & EDA 📊

An end-to-end Exploratory Data Analysis (EDA) and RFM (Recency, Frequency, Monetary) analysis of retail transactions. This project identifies high-value customer segments and regional sales performance using Python.

## 🚀 Project Overview
This analysis processes over 99,000 rows of retail data to:
- Identify revenue drivers across different shopping malls.
- Analyze demographic spending patterns (Age & Gender).
- Perform **RFM Segmentation** to categorize customers into 'Champions' and 'At-Risk' groups.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Google Colab / Jupyter Notebook

## 📈 Key Findings
- **Mall Performance:** Mall of Istanbul leads with 13.8M in total sales.
- **Customer Health:** Identified 5,425 "Champions" and 5,570 "At-Risk" customers using AI-driven ranking methods.
- **Payment Trends:** Cash remains the dominant payment method for the 40+ age demographic.
